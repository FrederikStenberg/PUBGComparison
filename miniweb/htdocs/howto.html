How to use

In order to use the PUBG comparer, insert a name into each of the text boxes on the front page.* For both players, select their respected region using the dropdown menu. Finally select for what month you would like to see the stats.** To proceed click the "Compare" button and wait for the page to find your desired statistics.

Example:

  • Insert "shroud" into the first player textbox.
  • Select "North America" as that players region.
  • Insert "lyxrN" into the second player textbox.
  • Select "Europe" as that players region.
  • Select February as the month.
  • Use the "Compare" button to view the comparison.

*Player names are CASE SENSITIVE.

**PUBG stats are based on seasons. Each month is equivalent to a season.

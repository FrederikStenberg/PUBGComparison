How to use:
1. Open miniweb and start miniweb.exe and a CMD-prompt should appear.
2. Type the IP adress shown in the CMD-prompt into any browser as the URL.
3. To understand how to use the web-page, please click the "i" icon on the front-page
	to get directed to a "How to" page.
